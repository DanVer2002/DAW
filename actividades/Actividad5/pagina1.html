Más etiquetas de HTML

Etiquetas: <dl></dl> <dt></dt> <dd></dd>

El elemento HTML dl representa una lista descriptiva. El elemento encierra una lista de grupos de términos (especificados con el uso del elemento dt) y de descripciones (proveídas con elementos dd). Algunos usos comunes para este elemento son implementar un glosario o para desplegar metadatos. Por ejemplo:

Escuela.
Unidad Profesional Interdisciplinaria de Ingenieria campus Tlaxcala.
Carrera
Ingeniera en Inteligencia Artificial.
Unidad de Aprendizaje.
Tecnologias para el desarrollo de aplicaciones Web.

Etiqueta: <blockquote></blockquote>

El elemento HTML blockquote sirve para citas en bloque. Por ejemplo:

En la paguina de desarrollo de Mozilla encontraras información sobre esta etiqueta.

Etiqueta: <q></q>

El elemento q para citas en línea. Por ejemplo:

Entonces el dijo Renovada será la hoja que ha sido rota.

Etiqueta: <cite></cite>

El elemento HTML cite marca una referencia a una fuente, o el autor de un texto citado. Por ejemplo:

Albert Einstein dijo: "La medida de la inteligencia es la capcidad de cambiar."

Etiqueta: <abbr></abbr>

El elemento HTML abbr representa una abreviación o acrónimo. Por ejemplo:

Estudio en el IPN

Etiqueta: <address></address>

El elemento HTML address aporta información de contacto. Por ejemplo:

Tal vez te podria interesar www.juntosporlamalintzi.com.
Puedes contactarme por email.
Daniel Vega Foundation
MX

Etiqueta: <sub></sub>

El elemento HTML sub define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice. Por ejemplo:

La fórmula química de agua es: H2O

Etiqueta: <sup></sup>

El elemento HTML sup define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice. Por ejemplo:

El número 4 también se puede representar como 22

Etiqueta: <code></code>

El elemento HTML code es el apropiado para marcar el código de un programa. Por ejemplo:

printf ("Hello World!\n");

Etiqueta: <pre></pre>

El Elemento HTML pre representa texto preformateado. El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo. Los espacios dentro de este elemento también son mostrados como están escritos. Por ejemplo:

   int main{
     int a;
     printf("Escribe un numero");
     scanf("%d",&n);
   }

Etiqueta: <var></var>

El elemento HTML var marca variables de programas y similares. Por ejemplo:

el bucle no se detendrá hasta que Variable sea igual a 1.

Etiqueta: <kbd></kbd>

El elemento HTML kbd marca el texto que debe introducir el usuario. Por ejemplo:

Si desea refrescar la página pulse F5

Etiqueta: <samp></samp>

El elemento HTML samp se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador. El contenido de esta etiqueta es renderizado generalmente usando la tipografía monoespaciada por defecto del navegador. Por ejemplo

Aparecerá una ventana con el mensaje: Bienvenido...

Etiqueta: <time></time>

El elemento HTML time representa un periodo específico en el tiempo. Puede incluir el atributo datetime para convertir las fechas en un formato interno legible por un ordenador, permitiendo mejores resultados en los motores de búsqueda o características personalizadas como recordatorios. Por ejemplo:

Yo nací el 30 de Diciembre en Tlaxcala, a las 10:00am.
